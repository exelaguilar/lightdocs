## Data and recovery

### Important paths

- `/.n8n/database.sqlite`
- `/.n8n/config`
- `/.n8n/nodes`
- `/opt/n8n.env`

### Backup procedure

- Stop n8n.service or use SQLite's online backup mechanism before copying /.n8n/database.sqlite and its WAL/SHM files.
- Preserve /.n8n/config and /opt/n8n.env as secrets; encrypted credentials may be unrecoverable without the matching encryption configuration.

### Observed state

- n8n data uses approximately 49 MiB.
- The system service starts with working directory /, so the default n8n state directory is /.n8n rather than /root/.n8n.
