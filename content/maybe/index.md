# Maybe

This section holds useful ideas that are not required for Lightdocs and should not distract from its local-first, microscopic deployment model.

:::callout type="info" title="Core stays local"
Everything under Maybe is optional. Reader rendering, Studio editing, SQLite indexing, static exports, built-in revisions, and Local Git work without these experiments.
:::

:::cards
:::card title="GitHub remote sync" href="/maybe/github-remote-sync"
Optionally publish a generated Git mirror to a hosted remote after Local Git needs are understood.
:::
:::
